No web server detected

FrameTrail needs PHP to run. In your FrameTrail folder, start PHP’s built-in server:

php -S localhost:8080

Then open http://localhost:8080 in your browser.

No PHP installed? Use XAMPP (Windows) or MAMP (Mac / Windows).

Alternatively, open index.html directly in Chrome or Edge — FrameTrail works without any server in those browsers.
